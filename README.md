## 🎯 학습 목표

이 저장소는 컴퓨터 과학의 핵심 개념을 체계적으로 정리하고, 빠르게 참조할 수 있도록 구성되었습니다.

### 주요 특징
- **명확한 구조**: 주제별로 폴더 구분
- **빠른 검색**: 목차와 링크를 통한 쉬운 네비게이션
- **핵심 정리**: 중요한 개념만 간결하게 정리
- **비교 표**: 개념 간 차이를 한눈에 파악
